\section{ISEMPTY Test For Variable Empty}

\subsection{Usage}

The \verb|isempty| function returns a boolean that indicates
if the argument variable is empty or not.  The general
syntax for its use is
\begin{verbatim}
  y = isempty(x).
\end{verbatim}
