\section{CLOCK Get Current Time}

\subsection{Usage}

Returns the current date and time as a vector.  The syntax for its use is
\begin{verbatim}
   y = clock
\end{verbatim}
where \verb|y| has the following format:
\begin{verbatim}
   y = [year month day hour minute seconds]
\end{verbatim}
