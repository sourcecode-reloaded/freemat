\section{FLIPLR Reverse the Columns of a Matrix}

\subsection{USAGE}

Reverses the columns of a matrix.  The syntax for its use is
\begin{verbatim}
   y = fliplr(x)
\end{verbatim}
where \verb|x| is matrix.  If \verb|x| is an N-dimensional array then
the second dimension is reversed.
