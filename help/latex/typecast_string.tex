\section{STRING Convert Array to String}

\subsection{Usage}

Converts the argument array into a string.  The syntax for its
use is
\begin{verbatim}
   y = string(x)
\end{verbatim}
where \verb|x| is an \verb|n|-dimensional numerical array.
