\section{FLIPUD Reverse the Columns of a Matrix}

\subsection{USAGE}

Reverses the rows of a matrix.  The syntax for its use is
\begin{verbatim}
   y = flipud(x)
\end{verbatim}
where \verb|x| is matrix.  If \verb|x| is an N-dimensional array then
the first dimension is reversed.
