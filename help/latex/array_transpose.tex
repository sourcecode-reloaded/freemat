\section{TRANSPOSE Matrix Transpose }

\subsection{Usage}

Performs a (nonconjugate) transpose of a matrix.  The syntax for
its use is
\begin{verbatim}
    y = transpose(x)
\end{verbatim}
and is a synonym for \verb|y = x.'|.
