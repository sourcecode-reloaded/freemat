\section{LOGICAL Convert to Logical}

\subsection{Usage}

Converts the argument to a logical array.  The syntax
for its use is
\begin{verbatim}
   y = logical(x)
\end{verbatim}
where \verb|x| is an \verb|n|-dimensional numerical array.  Any nonzero 
element maps to a logical 1.
