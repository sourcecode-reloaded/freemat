\section{FLIPDIM Reverse a Matrix Along a Given Dimension}

\subsection{USAGE}

Reverses an array along the given dimension.  The syntax for its use is
\begin{verbatim}
   y = flipdim(x,n)
\end{verbatim}
where \verb|x| is matrix, and \verb|n| is the dimension to reverse.
