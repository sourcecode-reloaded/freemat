\section{ASECD Inverse Secant Degrees Function}

\subsection{Usage}

Computes the inverse secant of the argument, but returns
the argument in degrees instead of radians (as is the case
for \verb|asec|. The syntax for its use is
\begin{verbatim}
   y = asecd(x)
\end{verbatim}
