\section{SQRT Square Root of an Array}

\subsection{Usage}

Computes the square root of the argument matrix.  The general
syntax for its use is
\begin{verbatim}
   y = sqrt(x)
\end{verbatim}
where \verb|x| is an N-dimensional numerical array.
